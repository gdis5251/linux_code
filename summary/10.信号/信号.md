# 信号--Signal

> 信号的作用就相当于神经系统，来帮助操作系统进程运行过程中遇到的各种意外。
>
> 在`Linux`系统中通过命令`kill -l`来查看所有信号。

## 信号产生的方式：

### 1. 通过终端按键产生信号（键盘产生）

**最常见的键盘产生的信号：**

`ctrl + c => 2`对应的是2号信号`SIGINT`。

`ctrl + z => 19`对应的是19号信号SIGSTOP。

`ctrl + \ => 3`对应的是3号信号SIGQUIT。
